ℹ️ 什么是 CPS 测试？

CPS（Clicks Per Second） 代表每秒鼠标点击次数。它是衡量你的手指反应、连点频率和游戏微操（例如 Minecraft 电竞玩家的 PVP 连招、即时战略游戏中的快捷操作）的一项关键指标。

📝 怎么开始测试？

  • 点击顶部导航栏的 “测试 / Test” 按钮进入测试主界面。
  • 在底部的控制面板中，自主选择适合你的测试时长（提供 1秒 / 3秒 / 5秒 / 10秒 / 30秒 / 60秒 多种规格）。
  • 将光标移入巨大的虚线测试框中，点击第一声即可自动触发倒计时。
  • 在剩余时间内，尽可能飞速地连续点击。结束后系统将自动计算出高精度的 CPS 报告，并存入本地历史记录。

💡 为什么我们更精准？

  • 毫秒级高精度：我们舍弃了传统的 `setInterval` 模糊计数，采用 Web API 中高精度的 `performance.now()` 时间戳追踪，精确还原每一次物理点击。
  • 智能段位体系：基于您的平均 CPS，系统将为您赋予从“乌龟”到“机械神族”的荣耀称号！
  • 本地持久化：所有测试记录均保存在您的本地浏览器中，刷新网页也不会丢失。
